Experiment No. 14: Data Visualization using Seaborn

Aim:

To perform data visualization using the Seaborn library in Python.

Objectives:

1.To understand advanced data visualization techniques.

2.To use Seaborn for statistical data visualization.

3.To create different types of plots using Seaborn.

4.To analyze patterns and relationships in data.

Theory:

Seaborn is a Python data visualization library built on top of Matplotlib. It provides a high-level interface for drawing attractive and informative statistical graphics.

Seaborn simplifies complex visualizations and provides built-in themes and color palettes. It is especially useful for visualizing relationships between variables.

Common plots in Seaborn include:

1.Line Plot – shows trends over time

2.Bar Plot – compares categories

3.Histogram – shows data distribution

4.Box Plot – shows spread and outliers

5.Heatmap – shows correlation between variables

Seaborn works well with Pandas DataFrames and makes data visualization more effective and visually appealing.

Procedure:

1.Import required libraries (pandas, seaborn, matplotlib.pyplot).

2.Load dataset using read_csv() or built-in datasets.

3.Use Seaborn functions to create plots:

3.1 Line plot using lineplot()

3.2 Bar plot using barplot()

3.3 Histogram using histplot()

3.4 Box plot using boxplot()

3.5 Heatmap using heatmap()

4.Add titles and labels to graphs.

5.Display plots using show().

Observations:

1.Line plot shows relationship between total bill and tip.

2.Bar plot compares total bill across different days.

3.Histogram shows distribution of total bill values.

4.Box plot identifies spread and outliers.

5.Heatmap shows correlation between numerical variables.

Result:

Various statistical plots were successfully created using Seaborn, and data patterns were effectively visualized.

Conclusion:

Seaborn is a powerful and user-friendly visualization library. It enhances the appearance of graphs and helps in better understanding of data relationships and patterns.
